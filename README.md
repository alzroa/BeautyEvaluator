# BeautyEvaluator

An open-source computer vision project to evaluate beauty based on the Marquardt Mask, facial symmetry, and body proportions.

## Features (Planned)
- [ ] Marquardt Mask overlay
- [ ] Facial Symmetry Analysis (Left vs Right)
- [ ] Body Proportions (Golden Ratio calculations)
- [ ] Image Landmark detection (using MediaPipe)

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py --image path/to/image.jpg
```
